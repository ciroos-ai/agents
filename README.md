# Ciroos Agents

Pre-built binary artifacts for Ciroos agents.

## Download

Download the latest release from [GitHub Releases](https://github.com/ciroos-ai/agents/releases).

## Release Artifacts

Each release publishes a tarball per supported architecture following the naming
convention:

```text
ciroos-agents-<platform>-<arch>-<version>.tgz
```

Examples:

| Platform | Architecture | Example                               |
|----------|--------------|---------------------------------------|
| linux    | amd64        | `ciroos-agents-linux-amd64-0.0.1.tgz` |
| linux    | arm64        | `ciroos-agents-linux-arm64-0.0.1.tgz` |
